## Build Setup

``` bash
# install dependencies
npm install

# Запуск эмулятора 
make emulator

# Установка пути при настройке эмулятора
make home

# Установка и сборка приложения на эмуляторе android
npm run android

# Установка и сборка приложения на эмуляторе ios
npm run ios

# Запуск приложения после установки
npm run start


```
